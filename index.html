how to:

Type in the top left or press the "a" button to make text. Currently, the following character set is supported, no numbers: abcdefghijklmnopqrstuvwxyz äöüß .,!?|_-

See how different values change the font by first selecting a property, then using the mouse wheel or arrow keys to increase or decrease. Then try some special modes via the dropdown. Many of them require either multiple rings or vertical stretch to actually have an effect. Use the checkboxes to toggle alternative characters or view modes.
